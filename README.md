# Site Checker

Автоматическая проверка доступности страниц СМУ, ИМП, МПЭ.

## Что проверяется

- 🏠 Главные страницы поддоменов
- 📁 Страница /catalog/
- 📂 Категории каталога
- 🏷️ Фильтры и теги
- 🛒 Карточки товаров (из sitemap.xml)
- 🔤 Битые переменные в текстах ({{city}}, %price%, undefined, [object Object])

## Запуск локально

```bash
pip install -r requirements.txt
streamlit run app.py
```

Откроется в браузере на http://localhost:8501

## Деплой на Streamlit Cloud

1. Залить этот репозиторий на GitHub
2. Зайти в https://streamlit.io/cloud
3. New app → выбрать репозиторий → Main file: `app.py`
4. Deploy

## Обновление каталогов

Каталоги хранятся в виде CSV-файлов в папке `catalogs/`:

- `{proj}-subdomains.csv` — поддомены (колонки: url, city)
- `{proj}-catalog.csv` — каталог категорий и фильтров (колонки: url, type)
- `{proj}-categories.csv` — отдельный файл актуальных категорий (только для СМУ)

Чтобы обновить — конвертировать новый xlsx в CSV через `convert_catalogs.py`
(нужны исходные xlsx из старого Node.js проекта в папке `/home/claude/site-checker/catalogs/`).

## Структура

```
app.py                — главный Streamlit-файл
sources.py            — парсинг CSV-каталогов + построение плана
http_checker.py       — async HTTP с ретраями
text_checker.py       — поиск битых переменных
sitemap.py            — загрузка sitemap.xml для товаров
history.py            — ротация выборки (7 дней)
profiles.py           — пресеты Быстрая / Стандартная / Полная
reporter.py           — формирование xlsx-отчёта
convert_catalogs.py   — разовая конвертация xlsx → CSV

catalogs/             — CSV-каталоги (читаются приложением)
projects/             — JSON-конфиги проектов
cache/                — кеш sitemap'ов и история ротации (не в git)
reports/              — сохранённые отчёты (не в git)
tests/                — тесты модулей
```

## Профили проверок

| Профиль | Городов | Категорий | Фильтров | Товаров | Время |
|---|---|---|---|---|---|
| Быстрая | 3 (Москва + 2) | 3 | 3 | 2 | ~1-2 мин |
| Стандартная | 6 (Москва + 5) | 5 | 5 | 3 | ~3-5 мин |
| Полная | 11 (Москва + 10) | 10 | 10 | 5 | ~10-20 мин |
| Свои настройки | вручную | вручную | вручную | вручную | — |

## Ротация выборки

Скрипт помнит какие URL проверялись за последние 7 дней. При формировании
случайной выборки им даётся меньший вес (0.3 от обычного), чтобы за неделю
постепенно покрывался весь каталог. История хранится в `cache/history-{proj}.json`.
