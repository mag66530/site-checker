# Автокликеры GSC и Яндекс.Вебмастер — инструкция

Эти инструменты **запускаются локально** на твоём ПК (не в облаке): они открывают
браузер на твоём компьютере и требуют ручного входа в Google/Yandex. Облачный
Streamlit так не умеет — сервер не управляет твоим браузером.

## Что делают

- **GSC** (`gsc_validate_fixes.py`): в Search Console по каждому ресурсу заходит в
  «Индексирование → Страницы», по причинам со статусом «Не начато» жмёт
  «Проверить исправление».
- **Вебмастер** (`webmaster_recheck.py`): по каждому сайту аккаунта заходит в
  диагностику и жмёт «Проверить» по ошибкам (ссылку «Проверьте» не трогает).

## Установка (один раз)

```
pip install -r requirements-local.txt
playwright install chromium
```

## Запуск через интерфейс (рекомендуется)

```
streamlit run launcher.py
```

Откроется локальная страница:
1. Выбери проект (МПЭ / СМУ / ИНП) — покажет, в какие аккаунты войти.
2. **Шаг 1** — «Открыть браузер для входа». Войди в нужные Google и Yandex
   аккаунты проекта. Окно не закрывай.
3. **Шаг 2 (GSC)** — обнови список ресурсов, затем запусти проверку исправлений.
4. **Шаг 3 (Вебмастер)** — запусти проверку ошибок.

Сначала пробуй с галкой **dry-run** (не кликает, только показывает что нашёл).
Убедился — снимай галку и запускай боевой.

## Выбор проекта = аккаунт входа

Каждый проект — это свой Google и Yandex аккаунт, и в нём уже заведены все домены
и поддомены этого проекта. Кликеры обходят все сайты/ресурсы того аккаунта, в
который ты вошёл. Поэтому «выбрать проект» = войти в нужный аккаунт.

## Запуск из терминала (без интерфейса)

```
# 1. Открыть браузер и войти
python gsc_save_session.py

# 2. GSC: собрать ресурсы и проверить исправления
python gsc_list_properties.py
python gsc_validate_fixes.py --dry-run        # проверка
python gsc_validate_fixes.py                  # боевой

# 3. Вебмастер
python webmaster_recheck.py --dry-run         # проверка
python webmaster_recheck.py                   # боевой
```

## Логи

- `gsc_validate_log.json`
- `webmaster_recheck_log.json`
- `gsc_properties.json` — список ресурсов GSC текущего аккаунта
