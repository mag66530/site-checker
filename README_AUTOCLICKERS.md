# Автокликеры GSC и Яндекс.Вебмастер - инструкция

Автокликеры встроены в приложение отдельной страницей **«Автокликеры»** (меню слева).

## Где работает

- **Локально** (приложение запущено на твоём ПК) - работает: открывает браузер на
  этом компьютере, ты логинишься, кликеры кликают.
- **Облако** (Streamlit Cloud по ссылке) - пока **не** работает: у сервера нет твоего
  браузера и нельзя пройти ручной вход Google/Yandex.
- **Свой сервер** (в планах) - заработает так же, как локально, если на сервере стоит
  Playwright/Chromium и однажды выполнен вход в нужные аккаунты. Код менять не надо.

## Что делают

- **ГСК** (`gsc_validate_fixes.py`): по каждому ресурсу заходит в «Индексирование →
  Страницы», по причинам со статусом «Не начато» жмёт «Проверить исправление».
- **Вебмастер** (`webmaster_recheck.py`): по каждому сайту аккаунта в диагностике
  жмёт «Проверить» по ошибкам (ссылку «Проверьте» не трогает).

## Установка с нуля (Windows, для нетехнических людей)

**1. Поставить Python** - https://www.python.org/downloads/
На первом экране установщика галка **«Add Python to PATH»**, потом «Install Now».

**2. Поставить Google Chrome** (если нет) - https://www.google.com/chrome/

**3. Скачать файлы проекта**
GitHub репозитория → зелёная кнопка **Code → Download ZIP** → распаковать (напр. на рабочий стол).

**4. Открыть терминал в папке проекта**
Зайти в папку проекта в проводнике → в адресной строке напечатать `powershell` → Enter.

**5. Установить зависимости (один раз):**
```
pip install -r requirements-local.txt
playwright install chromium
```

**6. Запустить приложение:**
```
streamlit run app.py
```

## Как пользоваться

1. В меню слева открой **«Автокликеры»**.
2. Выбери **проект** (МПЭ / СМУ / ИНП) - покажет, в какие аккаунты войти.
3. **Шаг 1** → «Открыть браузер для входа». В открывшемся Chrome войди в Google и
   Yandex аккаунты проекта. Окно не закрывай.
4. **Шаг 2** → отметь галочки «Прокликать ошибки ГСК» и/или «Прокликать ошибки
   Вебмастера». Для ГСК сначала нажми «Обновить список ресурсов ГСК».
5. Оставь галку **dry-run** (проверка без кликов), нажми «Запустить выбранное».
   Убедился, что находит правильно - сними dry-run и запусти боевой.

## Запуск из терминала (без интерфейса)

```

python gsc_list_properties.py         # собрать ресурсы GSC
python gsc_validate_fixes.py --dry-run

python webmaster_recheck.py --dry-run
python webmaster_recheck.py
```
