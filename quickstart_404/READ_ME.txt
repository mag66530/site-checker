ПРОВЕРКА СТРАНИЦ В ИНДЕКСЕ НА 404 - быстрый запуск
====================================================

Важно: чёрное окно (консоль) будет писать шаги на английском - это
специально, чтобы русские буквы не ломались в Windows-консоли. Вот
что означает каждый шаг:

  STEP 1 of 3: opening Chrome
  = Шаг 1 из 3: открывается Chrome

  STEP 2 of 3: log into Yandex Webmaster
  = Шаг 2 из 3: войдите в Яндекс.Вебмастер

  STEP 3 of 3: safe test run
  = Шаг 3 из 3: пробный (безопасный) запуск проверки

  DONE. Copy ALL the text above...
  = ГОТОВО. Скопируйте весь текст выше...

Как запускать:

1) Распакуйте архив ПРЯМО В папку проекта site-checker - так,
   чтобы файл webmaster_404_export.py оказался РЯДОМ с app.py и
   gsc_save_session.py (не в отдельной новой папке типа Downloads).

2) Двойной клик по START.bat.

3) Скрипт сам всё проведёт по шагам:
   - откроет отдельный Chrome (STEP 1),
   - попросит вас войти в Яндекс на webmaster.yandex.ru в этом же
     Chrome и нажать любую клавишу, когда будете готовы (STEP 2),
   - запустит пробную (безопасную, ничего не скачивающую) проверку
     (STEP 3).

4) В конце скопируйте ВЕСЬ текст из чёрного окна и пришлите его в
   чат - по нему поправим, если что-то не так.

Если чёрное окно сразу пишет "ERROR: autoclick_browser.py not found" -
значит файлы распакованы не в ту папку, см. пункт 1 (нужно ВНУТРИ
папки проекта, не в Downloads).
