ПРОВЕРКА СТРАНИЦ В ИНДЕКСЕ НА 404 - быстрый запуск
====================================================

Перед тем как эта папка появится у вас - один раз (уже один раз надо
было это сделать) откройте терминал/PowerShell в папке проекта и
выполните:

    git fetch origin claude/form-security-hardening-yf1x50
    git checkout claude/form-security-hardening-yf1x50

После этого папка quickstart_404 появится сама. Дальше - без единой
команды, только двойной клик по файлам ПО ПОРЯДКУ:

1) "1-открыть-браузер-для-входа.bat"
   Откроется отдельный Chrome. Зайдите там на webmaster.yandex.ru
   и войдите в Яндекс (если ещё не вошли). Окно и Chrome не
   закрывайте.

2) "2-проверить-404-пробный-запуск.bat"
   Пробный, безопасный запуск - ничего не скачивает и никуда не
   звонит, только смотрит, открывается ли нужная страница.
   В конце появится текст лога - скопируйте его ВЕСЬ и пришлите.

Если на шаге 2 что-то пойдёт не так - скрипт сам напечатает, какие
кнопки/ссылки он видит на странице Вебмастера. Это нормально при
первом запуске - по этому логу поправим один раз, и дальше будет
работать само.

После того как шаг 2 пройдёт успешно - будет третий файл (пришлю
следующим шагом) уже для настоящего скачивания и проверки.
